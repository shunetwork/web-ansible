# Cisco 网络设备自动化管理平台

基于 **Django Jet** + **SQLite** + **Ansible** 的 Cisco 网络设备集中管理与自动化运维平台，提供：

- 设备集中管理与分组
- 模板化配置下发（Jinja2）
- 批量命令执行与异步任务（Celery + Redis）
- 配置备份与差异分析
- 用户权限管理与操作审计
- Web 可视化后台操作（Django Jet）

---

## 功能模块

1. **设备管理**
   - 添加、删除、修改设备信息
   - SSH 测试连通性
   - 自动识别设备型号与版本
   - 批量导入/导出 CSV

2. **模板管理**
   - Jinja2 模板支持变量参数化
   - 模板版本控制
   - Web 界面预览渲染

3. **任务执行**
   - 批量下发模板
   - 自定义命令执行
   - 任务实时状态及日志输出

4. **配置备份**
   - 自动定时备份 running-config
   - 配置差异对比
   - 一键恢复上版本

5. **用户与权限**
   - Django 内置用户系统
   - 管理员/运维/只读角色
   - 操作日志记录

6. **异步调度**
   - Celery + Redis 异步执行任务
   - 定时备份和健康检查
